## Introduction

Hi! I'm ***Restart/Hirutonbi***!! (the latter comes from the song of the same name by Yorushika, meaning "Daytime Thief")!! I'm from NUAA, majoring in Electronic Information Engineering. This blog is used to document my personal life and share my favorite music, illustrations, articles, and more.

Based on the [Momo](https://github.com/Motues/Momo) project, this blog has been extended with many new features using **vibe coding**. If you encounter any issues during your experience, feedback is welcome.

## New Features

- **Casual Thoughts**: a local micro-blog / timeline based on Astro Content Collections, zero backend dependencies, Markdown writing, with automatic card folding.
- **Appreciation System**: a standalone appreciation page integrating gallery and article appreciation, supporting multi-level tag classification, AND/OR filtering, and dedicated archives.
- **Archive Enhancement**: separate archives for blog and appreciation, dimension-based tag grouping, tag tree, first-letter grouping, collapsible/expandable sections, and scrollable areas.
- **Multi-Level Tags**: a unified `dimension:value` tag format across the entire site, with sidebar grouping by dimension and sorting by first letter.
- **Image Preview**: integrated PhotoSwipe Lightbox for fullscreen preview, supporting scroll zoom, panning, left/right navigation, and close button.
- **Comment Enhancements**: carousel navigation for images in comments, emoji support, and pagination optimizations.
- **Music Player**: a global music player with playlists deployed on Vercel using Netease Cloud Music API, with reverse proxy to mitigate blocking.
- **Rotating Background**: supports multiple background images in a carousel, adapted to light/dark themes.
- **Custom Cursor**: adds a Lilith custom cursor, with new click and trailing effects.
- **R2 Image Hosting**: gallery and background images migrated to Cloudflare R2 with custom domain CDN, reducing repo size
